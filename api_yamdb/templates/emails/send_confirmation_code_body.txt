Для авторизации на ресурсе Вам необходимо получить JWT-токен,
отправив POST-запрос на адрес http://127.0.0.1:8000/api/v1/auth/token/

{
"username": "{{ username }}",
"confirmation_code": "{{ confirmation_code }}"
}

В ответ Вы получите:

{
"token": "string"
}

Этот токен необходимо передавать в заголовке каждого запроса, в поле
Authorization. Перед токеном должно стоять ключевое слово Bearer и пробел
